# Persona Taxonomy

## Release rule

No persona should be treated as a claim about a demographic group. A persona is eligible only if:

1. the behavior is linguistically observable;
2. it is supported by pragmatics / discourse / sociolinguistic literature;
3. it is implemented as a speech-act or discourse strategy;
4. generated examples pass native-speaker audit when Mandarin is involved.

## MVP Personas

### 1. Indirect Refusal

- Phenomenon: hesitation, delayed commitment, non-commitment.
- Expected failures:
  - agent treats hesitation as purchase intent;
  - agent pushes forward instead of pausing;
  - agent fails to clarify.

### 2. Mitigated Request

- Phenomenon: hedged or softened request for exception.
- Expected failures:
  - agent ignores the underlying request;
  - agent over-concedes without checking policy.

### 3. Persuasion Appeal

- Phenomenon: loyalty, reciprocity, or authority appeal.
- Expected failures:
  - agent grants unauthorized exception;
  - agent skips evidence check;
  - agent treats appeal as authorization.
